# ベッドスケジュール管理

## この機能でできること
*   透析室のベッドごとに、どの患者さんがいつ入室するかをスケジュール管理できます。
*   カレンダー上で空いているベッドを探して、簡単に予約を入れることができます。
*   急な変更があった場合、ドラッグ操作で予約を移動できます。

## 詳細操作ガイド

### スケジュールを確認する
画面上部にあるボタンで、表示期間を切り替えられます。
*   **日表示**: 1日の詳細なスケジュールを時間単位で確認できます（画面左上の「今日」や「日」ボタンを使用）。
*   **週表示**: 1週間分のスケジュールをまとめて確認できます。

### 新しく予約を入れる
1.  スケジュール表の中で、予約を入れたい「空いている時間枠」をクリックします。
2.  「治療スケジュールの作成」という画面が開きます。
3.  **患者選択**: 「患者名」の欄で、対象の患者さんを選びます（検索もできます）。
4.  **日時確認**: 開始日時・終了日時が合っているか確認します。
5.  **保存**: 右下の「保存」ボタンを押すと、予約が作成されます。

### 予約を変更・移動する
*   **時間の変更（ドラッグ）**:
    *   予約のバーをマウスで掴んで（クリックしたまま）、別の時間や別のベッドへ動かすことができます。
*   **詳細の変更（編集）**:
    *   変更したい予約をクリックします。
    *   編集画面が開くので、内容を修正して「保存」を押します。

### 予約を削除する
1.  削除したい予約をクリックして編集画面を開きます。
2.  画面内にある「削除」ボタン（ゴミ箱アイコン）を押します。
3.  確認メッセージが出るので、問題なければ削除を実行してください。

## 入力のポイント
*   **患者検索**: 患者名の一部を入力すると候補が出てきます。
*   **重複チェック**: すでに予約が入っている時間に重ねて予約しようとすると、エラーになるか、警告が出ることがあります。空いている場所を選んでください。
